# Changelog

Все значимые изменения проекта RunTrack фиксируются здесь.
Формат версий: [SemVer](https://semver.org/) — `0.MINOR.PATCH`

- **MINOR** растёт при новых фичах
- **PATCH** растёт при багфиксах

---

## [0.4.0] — Sprint 4 (текущий)

### Исправлено

- `JSON.parse` без try/catch в `storageService` → крэш при повреждённых данных в AsyncStorage
- Стейт-гонка `gps.distance` при остановке забега — дистанция теперь пересчитывается из `coordsRef`
- `resumeTracking()` не проверял разрешение GPS — крэш если пользователь отозвал его в фоне
- Таймер стартовал раньше проверки разрешения GPS — при отказе тикал без GPS
- Сглаживание темпа — три уровня защиты от GPS-шума: диапазон [2:00–20:00/км], фильтр выбросов ×3, скользящее среднее по 5 точкам
- Валидация Supabase env-переменных с понятной инструкцией в консоли
- Дата забега в истории = момент нажатия СТАРТ, а не момент сохранения

### Добавлено

- Onboarding экран (3 слайда, горизонтальный свайп, точки-индикаторы, кнопка «Пропустить»)
- Флаг `@runtrack_onboarding_complete` в AsyncStorage — показывается только при первом запуске

### Удалено

- Мёртвый код: функция `replaceAllRuns` в `storageService`

## [0.3.0] — Sprint 3

### Добавлено

- Supabase Auth (Magic Link — вход по email без пароля)
- Синхронизация пробежек с облаком (upload / download / merge)
- Шеринг карточки забега в PNG (`react-native-view-shot` + `expo-sharing`)
- Компонент `ShareCard` (360×480px, дистанция / время / темп / брендинг)
- Русская локализация через i18n (`i18next` + `react-i18next`)
- Все тексты вынесены в `src/i18n/ru.json`

---

## [0.2.0] — Sprint 2

### Добавлено

- GPS-трекинг через `expo-location` (хук `useGPSTracker`)
- Экран итогов (`SummaryScreen`): карта маршрута, сплиты по км, калории, шеринг
- `AsyncStorage` — сохранение / загрузка / удаление пробежек (`storageService`)
- История пробежек (`HistoryScreen`): список, суммарная статистика, свайп-удаление
- Режим просмотра итогов из истории (без кнопки «Сохранить»)
- Tab-навигация: 🏃 Бег / 📊 История / 📈 Статистика
- Экран статистики (`StatsScreen`): барчарт дистанции, линейный график темпа, личные рекорды
- `victory-native` v36 для SVG-графиков

---

## [0.1.0] — Sprint 1

### Добавлено

- Expo-проект с TypeScript (strict mode)
- Главный экран (`HomeScreen`): таймер, GPS-статистика, кнопки управления
- Анимированная кнопка СТАРТ / ПАУЗА / ПРОДОЛЖИТЬ (spring-анимация, 120px)
- Хук `useRunTimer` — точный таймер с паузой и сбросом
- `locationService`: haversine, расчёт темпа, форматирование времени / дистанции
- Базовый UI: фон `#0D0D0D`, акцент `#00E5A0`, тёмная тема
