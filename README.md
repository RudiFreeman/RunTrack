## Для кого это приложение

RunTrack создан для бегунов в России и СНГ, которым нужен **простой и быстрый трекер** без лишних подписок и сложных настроек. Открыл — нажал СТАРТ — побежал.

## Технологии

- **React Native** — кроссплатформенный UI (iOS + Android)
- **Expo SDK** — быстрая разработка и деплой
- **TypeScript** — строгая типизация, без `any`
- **Supabase** — PostgreSQL в облаке, история пробежек, авторизация
- **expo-location** — GPS-трекинг в реальном времени
- **AsyncStorage** — офлайн-кэш пробежек

## Установка и запуск

```bash
# Клонировать репозиторий
git clone https://github.com/RudiFreeman/RunTrack.git
cd RunTrack

# Установить зависимости
npm install

# Запустить
npx expo start

# На телефоне: открыть Expo Go и отсканировать QR-код
```
