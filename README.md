## Для кого это приложение

RunTrack создан для бегунов в России и СНГ, которым нужен **простой и быстрый трекер** без лишних подписок и сложных настроек. Открыл — нажал СТАРТ — побежал.

## Технологии

- **React Native** + **Expo SDK 51** — кроссплатформенный UI (iOS + Android)
- **TypeScript** — строгая типизация, без `any`
- **expo-location** — GPS-трекинг в реальном времени
- **react-native-maps** — карта маршрута на экране итогов
- **AsyncStorage** — офлайн-хранилище истории пробежек
- **victory-native** — графики дистанции и темпа
- **React Navigation** — Tab + Stack навигация

## Установка и запуск

```bash
# Клонировать репозиторий
git clone https://github.com/RudiFreeman/RunTrack.git
cd RunTrack

# Установить зависимости
npm install

# Запустить
npx expo start

# На телефоне: открыть Expo Go и отсканировать QR-код
```

## Структура проекта

```
RunTrack/
├── App.tsx                          # Точка входа, Tab + Stack навигация
├── src/
│   ├── components/
│   │   ├── HomeScreen.tsx           # Экран бега (таймер, GPS, кнопки)
│   │   ├── StartButton.tsx          # Анимированная кнопка СТАРТ/ПАУЗА
│   │   └── StatCard.tsx             # Карточка: значение + подпись
│   ├── hooks/
│   │   ├── useRunTimer.ts           # Точный таймер с паузой
│   │   └── useGPSTracker.ts         # GPS-трекинг через expo-location
│   ├── navigation/
│   │   └── types.ts                 # Типы навигации (Tab, Stack)
│   ├── screens/
│   │   ├── SummaryScreen.tsx        # Итоги забега: карта, сплиты, калории
│   │   ├── HistoryScreen.tsx        # История забегов со свайпом удаления
│   │   └── StatsScreen.tsx          # Статистика: графики, личные рекорды
│   ├── services/
│   │   ├── locationService.ts       # haversine, расчёт темпа, форматирование
│   │   └── storageService.ts        # CRUD для AsyncStorage
│   └── types/
│       └── index.ts                 # SavedRun, RunSession, Coordinate, Split
```
