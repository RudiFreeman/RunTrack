# RunTrack — Беговое приложение

MVP бегового трекера для рынка СНГ, построенное на Expo + TypeScript.

## Стек технологий

- **Expo SDK** (blank-typescript шаблон)
- **React Native** — кроссплатформенный UI (iOS + Android)
- **TypeScript** — строгая типизация

## Структура проекта

```
src/
├── components/    # UI-компоненты (HomeScreen, StartButton, StatCard)
├── hooks/         # Кастомные хуки (useRunTimer)
├── services/      # Бизнес-логика (расчёт дистанции, темпа, форматирование)
└── types/         # TypeScript-типы (RunSession, RunStatus, Coordinate)
```

## Запуск

```bash
npm install
npx expo start
```

## Основной функционал (MVP)

- Кнопка СТАРТ/ПАУЗА/ПРОДОЛЖИТЬ с анимацией
- Таймер пробежки в реальном времени
- Отображение дистанции и темпа (мин/км)
- Тёмный интерфейс (#0D0D0D) с акцентным зелёным (#00E5A0)
- Интерфейс полностью на русском языке
